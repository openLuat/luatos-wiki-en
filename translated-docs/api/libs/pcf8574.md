**Example**

```lua
--Note: due to the use of sys.wait() all apis need to be used in the coroutine
-- Usage Examples
local pcf8574 = require "pcf8574"
i2cid = 0
i2c_speed = i2c.FAST
sys.taskInit(function()
    i2c.setup(i2cid,i2c_speed)
    pcf8574.init(i2cid)--initialization, passing in i2c_id
    for i=0,7 do
        print(pcf8574.pin(i))
    end
    pcf8574.pin(0,1)
    for i=0,7 do
        print(pcf8574.pin(i))
    end
end)

```
